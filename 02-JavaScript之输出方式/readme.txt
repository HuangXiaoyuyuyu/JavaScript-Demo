在JavaScript中，输出到页面或者控制台有以下几种方式：
1.以弹窗的形式出现：alert("AAAAA");
2.以弹窗的形式出现并且可以进行数据携带： prompt("请输入你的名字：")；
3.以弹窗的形式出现并且可以进行选择：confirm("EEEEE");
4.在控制台以日志的形式出现： console.log("BBBBB");
5.在控制台以警告形式出现： console.warn("CCCCC");
6.在控制台以错误显示出现：console.error("DDDDD")；

注意点：alert();有断点效果  因为需要按了之后才会继续执行 具体事例可以看本次Demo
			
另外以上所有的语句都需要在<script></script>标签中书写 如有疑问 请留言
